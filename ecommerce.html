Growque E-commerce Platform

A complete e-commerce platform for a Saudi brand, featuring product catalog management, voucher systems, order tracking, and customer account management. The system includes a secure admin dashboard for streamlined operations.

Built using Python (Django), HTML, CSS, JavaScript, Bootstrap, and PostgreSQL, the solution enables administrators to manage products, monitor sales, and handle customer interactions efficiently.

Key Features:
  • Product & Category Management

    Admins can add, edit, delete, and organize products with images and dynamic status control.

  • Voucher Management

    Create time-limited vouchers with usage restrictions and customer-specific rules.

  • Order Processing & Tracking

    Track orders from placement to delivery, with notifications for each status update.

  • Customer Dashboard

    Customers can manage profiles, view past orders, and maintain wishlists.

  • Secure Payments

    Integrated payment gateway handling for smooth transactions.

  • Reporting

    Sales analytics, voucher redemption reports, and customer activity logs.

Technical Stack:
Python Django HTML CSS JavaScript Bootstrap PostgreSQL
